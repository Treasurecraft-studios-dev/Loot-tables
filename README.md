Windows:

1. Download the .jar file from this post.

2. Press Windows + R, type %appdata%\.minecraft\mods and hit Enter.

Mac:

1. Download the .jar file from this post.

2. Open Finder and press Command + Shift + G.

3. Type: ~/Library/Application Support/minecraft/mods and hit Enter.

4. Place the .jar file into the "mods" folder.

5. Launch Minecraft with Forge/Fabric 1.19.2 installed.

Linux:

1. Download the .jar file from this post.

2. Open your home directory and press Ctrl + H to show hidden folders.

3. Navigate to: ~/.minecraft/mods

4. Place the .jar file into the "mods" folder.

5. Launch Minecraft with Forge/Fabric 1.19.2 installed.
